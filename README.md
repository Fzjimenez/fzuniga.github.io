# Franki Zúñiga Jiménez - Portfolio

Portafolio web profesional estilo hacker cyber para mostrar proyectos de Cloud, Azure, Microsoft Security, SIEM e infraestructura híbrida.

## Cómo verlo localmente

Solo abre `index.html` en tu navegador.

## Cómo publicarlo en GitHub Pages

1. Crea un repositorio en GitHub.
   - Recomendado: `fzuniga.github.io` si tu usuario es `fzuniga`.
   - Alternativo: `portfolio`.

2. Sube estos archivos al repositorio:
   - `index.html`
   - `styles.css`
   - `script.js`
   - carpeta `assets`

3. En GitHub:
   - Ve a **Settings**
   - Luego **Pages**
   - En **Build and deployment**, selecciona:
     - Source: `Deploy from a branch`
     - Branch: `main`
     - Folder: `/root`
   - Guarda los cambios.

4. Tu web quedará publicada en:
   - Si el repo es `fzuniga.github.io`: `https://fzuniga.github.io`
   - Si el repo es `portfolio`: `https://fzuniga.github.io/portfolio`

## Personalización rápida

Edita en `index.html`:
- Correo
- LinkedIn
- GitHub
- Proyectos
- Skills
- Descripción profesional

## Nota profesional

Antes de publicar, evita incluir:
- Nombres reales de clientes
- IPs públicas o privadas
- Dominios internos
- Capturas con datos sensibles
- Credenciales o tokens

La gracia del portafolio es mostrar criterio técnico, no regalarle inteligencia al atacante.
